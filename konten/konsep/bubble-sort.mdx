# Bubble Sort

Bubble Sort adalah algoritma pengurutan berbasis perbandingan. Algoritma ini memiliki kompleksitas waktu rata-rata dan terburuk sebesar $O(n^2)$, di mana $n$ adalah jumlah item yang diurutkan.

Meskipun tidak efisien untuk dataset besar, algoritma ini sederhana untuk dipahami dan diimplementasikan.
