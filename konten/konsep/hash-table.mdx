# Hash Table

Hash table menggunakan fungsi hash untuk menghitung indeks, juga disebut *hash code*, ke dalam array *bucket* atau *slot*, dari mana nilai yang diinginkan dapat ditemukan.

Operasi pencarian, penyisipan, dan penghapusan rata-rata memiliki kompleksitas waktu $O(1)$, menjadikannya sangat efisien.
