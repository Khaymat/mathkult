# Linked List

Sebuah linked list terdiri dari node-node di mana setiap node berisi data dan referensi (tautan) ke node berikutnya dalam urutan.

Operasi seperti penyisipan dan penghapusan bisa lebih efisien daripada array karena tidak perlu menggeser elemen. Namun, akses acak tidak efisien dengan kompleksitas $O(n)$.
