# Binary Search

Binary search bekerja dengan berulang kali membagi separuh dari bagian daftar yang dapat berisi item, sampai Anda mempersempit lokasi yang mungkin menjadi hanya satu.

Algoritma ini memiliki kompleksitas waktu $O(\\log n)$, yang jauh lebih cepat daripada pencarian linier $O(n)$ untuk array besar.
